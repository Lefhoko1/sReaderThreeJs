# Roadmap & Milestones

## Phase 0 — Foundations
- Docs (vision, requirements, domain model, architecture). ✔
- Choose backend (Supabase vs NestJS+Postgres). PoC sync.

## Phase 1 — App Skeleton
- Integrate React Native Paper theme and base navigation.
- Entity definitions (TS + Zod) and SQLite schema scaffolding.
- Repository interfaces; local storage for Assignments, Schedules, Friends.

## Phase 2 — Assignments & Schedules
- CRUD for assignments (tutor) and schedules (student) with visibility rules.
- Offline-first sync and conflicts.

## Phase 3 — Multiplayer MVP
- Lobby, turns, reactions; room lifecycle.

## Phase 4 — Guardian Engagement & Rewards
- Rewards assignment, progress views, notifications.

## Phase 5 — Payments & Subscription
- Orange Money integration; tickets ledger.

## Phase 6 — Hardening
- Security, analytics, accessibility, perf.
